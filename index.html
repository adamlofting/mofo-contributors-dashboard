NUMBER OF CONTRIBUTORS (by contribution type)

This breakdown of contribution types is to provide a quick insight into the Total Active number shown in the graph above. Not all 'types' are relevant to all teams.

Loading via Code & Bugs

Including: commits merged into Github repos and filing bugs on Bugzilla or Github.

Loading via Localization

Including: localisation via Transifex.

Loading via Content

Including: localisation via Transifex, guest blogging on MoFo sites, creating teaching kits and curriculum and creation of other ad-hoc high value content.

Loading via Events

Including: Hosting a Webmaker event (e.g. Maker Party); instructing or volunteering at Software Carpentry Bootcamps; hosting, volunteering or participating in a session at MozFest.

Loading via Completed Training

Including: Webmaker Mentors and Super Mentor who have completed the training and 'got the badge'.

Loading via Community Participation

Hive Community Badges and Ad-hoc contribution logged manually. Including: People who present on community calls and the individuals who help us form important partnships.

Loading via Testing & QA

Ad-hoc contribution logged manually. Including: Attending and running face-to-face user testing sessions.

Loading via Build on our APIs

Ad-hoc contribution logged manually. Including: Integrating with the MakeAPI.
